IComeFromTheNet's PEAR Channel Server

Registering the channel: 
pear channel-discover icomefromthenet.github.com/pear
Listing available packages: 
pear remote-list -c icomefromthenet
Installing a package: 
pear install icomefromthenet/package_name
Installing a specific version/stability: 
pear install icomefromthenet/package_name-1.0.0
pear install icomefromthenet/package_name-beta
Receiving updates via a feed: 
http://icomefromthenet.github.com/pear/feed.xml
